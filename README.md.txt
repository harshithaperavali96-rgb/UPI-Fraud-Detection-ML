# UPI Fraud Detection using Machine Learning

## Project Description
This project detects fraudulent UPI transactions using Machine Learning techniques. It analyzes transaction details and predicts whether a transaction is genuine or fraudulent.

## Features
- Fraud Detection
- Data Preprocessing
- Machine Learning Model
- Accuracy Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Future Scope
- Real-time Fraud Detection
- Deep Learning Integration
- Banking API Integration
